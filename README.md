## Features

- **Bilingual content** — Chinese explanations with English mathematical terminology
- **Interactive visualizations** — Canvas 2D simulations with draggable points, sliders, and animations
- **Worked examples** — step-by-step solutions for every core concept
- **Exercises with hints** — progressive difficulty from computational to proof-based
- **Monte Carlo simulations** — CLT, bootstrap, hypothesis testing brought to life

## Quick Start

# Clone the repository
